# Carlos Tancler
Este site tem como objetivo informar a comunidade escolar.

## Sobre
O site original da escola foi feito usando um único template do Google Sites e, por esse motivo, ele possui muita informação na página inicial, o que acabou gerando uma interface muito poluída visualmente. Pensando nisso, eu decidi criar esse site.

Inicialmente, esse site seria feito usando o ReactJS, uma biblioteca para o JavaScript que ajuda a criar interfaces gráficas utilizando componentes isolados. Eu acabei não utilizando esse framework, pois ele seria muito overkill e demoraria muito tempo para organizar tudo, então, no lugar dele, decidi utilizar um framework mais simples, o EJS.

O EJS é uma biblioteca que permite gerar sites estáticos em HTML através de códigos JavaScript. O JS é executado durante o desenvolvimento, mais ou menos como faz o Webpack, e por esse motivo não há um impacto na performance do site.

Esse perfil pertence à Escola Carlos Tancler, e esse repositório contém apenas a build do site. O script que gerou essas páginas estáticas está [neste repositório](https://www.github.com/emptyvoxel/school-site), no meu perfil principal.

Se encontrar algum problema na minha biblioteca, sinta-se livre para abrir um issue por lá, ou mesmo um pull request com a sua fix.
